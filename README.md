# Игра **Найди Трио**

Игра похожа на классическую игру **Найди пару** или **Мемори**.
Суть игры заключается в нахождении **трех** похожих картинок подряд.
В сложном режиме необходимо так же найти три похожих картинки, но при совершении ошибки все найденые картинки будут снова "закрыты".

Рекорды записываются в localStorage
